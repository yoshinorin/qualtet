> [Live (Qualtet is a backend of this site)](https://yoshinorin.net)

An example of architecture.

## Requirements

* sbt 1.9.x
* Scala 3.3.x
* Java 17.x
* MariaDB 10.11.x
* docker & docker-compose 3.x (for test)
